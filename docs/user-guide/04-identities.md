# Identities

PassKey can store personal identity information — useful for quickly filling in registration forms or keeping a secure record of your identity documents.

---

## Viewing Your Identities

The **Identities** section displays a grid of identity cards, each showing:

- **Avatar** — A circular badge with the first letter of the name
- **Full name**
- **Email address**
- **Phone number**
- **City**

---

## Adding an Identity

1. Press **Ctrl+N** or click the **+** button.
2. The detail view has three expandable sections:

### Personal Data

| Field | Description |
|-------|-------------|
| First name | Given name |
| Last name | Family name |
| Date of birth | Selected via a calendar date picker |
| Email | Primary email address |
| Phone | Phone number (e.g., +39 333 1234567) |

### Postal Address

| Field | Description |
|-------|-------------|
| Street | Street name and number |
| City | City or town |
| Postal code | ZIP or postal code |
| Province | Province, county, or state |
| Region | Region (where applicable) |
| Country | Country name |

### Identity Documents

You can store details for up to four document types:

| Document | Fields |
|----------|--------|
| National ID card | Number, issue date, expiry date |
| Health card | Number, issue date, expiry date |
| Driver's licence | Number, issue date, expiry date |
| Passport | Number, issue date, expiry date |

3. Fill in the sections you need — all fields are optional.
4. Click **Save**.

> **Tip:** When editing, the sections with data automatically expand so you can quickly see and modify them.

---

## Editing an Identity

Select an identity and press **F2**, or click to open the detail panel and then click the edit button. All three sections become editable.

---

## Deleting an Identity

Select an identity and press **Delete**, or use the delete button in the detail panel. A confirmation dialog will appear.

---

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+N | Create new identity |
| Ctrl+F | Focus the search box |
| F2 | Edit the selected identity |
| Delete | Delete the selected identity (with confirmation) |
| Esc | Close the detail panel |
| Ctrl+3 | Navigate to Identities section |
