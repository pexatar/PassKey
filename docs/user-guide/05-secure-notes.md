# Secure Notes

PassKey includes an encrypted note-taking feature for storing sensitive text information that does not fit into the password, card, or identity categories.

---

## Viewing Your Notes

The **Secure Notes** section displays notes in a list, each showing:

- **Category icon** — A Fluent icon representing the note category
- **Title**
- **Category label**
- **Colour indicator** — A pastel accent colour

---

## Adding a Note

1. Press **Ctrl+N** or click the **+** button.
2. Fill in the fields:

| Field | Description | Required |
|-------|-------------|----------|
| Title | A descriptive name for the note | Yes |
| Category | One of 10 predefined categories | Yes |
| Colour | A pastel colour from the palette | No |
| Content | The note text (multi-line) | No |

3. Click **Save**.

---

## Categories

Each note is assigned to one of 10 categories, each with a distinctive Fluent icon:

| Category | Icon | Typical use |
|----------|------|-------------|
| Personal | Person | Personal information |
| Work | Briefcase | Work-related notes |
| Finance | Money | Financial records, account details |
| Medical | Heart pulse | Medical information, prescriptions |
| Legal | Gavel | Legal documents, contracts |
| Travel | Airplane | Travel itineraries, booking references |
| Education | Graduation cap | Academic records, course notes |
| Technology | Laptop | Software licences, API keys, configs |
| Social | People | Social media accounts, community info |
| Other | Note | Anything else |

---

## Note Colours

Notes can be assigned a pastel colour from a dedicated palette (separate from the credit card colours). The colour appears as an accent in the note list and detail views.

---

## Editing a Note

Select a note and press **F2**, or click to open the detail panel and click the edit button.

---

## Deleting a Note

Select a note and press **Delete**, or use the delete button in the detail panel. A confirmation dialog will appear.

---

## Searching

Use **Ctrl+F** to focus the search box. Search matches against the note title and content (full-text search).

---

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+N | Create new note |
| Ctrl+F | Focus the search box |
| F2 | Edit the selected note |
| Delete | Delete the selected note (with confirmation) |
| Esc | Close the detail panel |
| Ctrl+4 | Navigate to Secure Notes section |
